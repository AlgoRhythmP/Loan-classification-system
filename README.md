# Loan Classification System

## Project Overview

A machine learning based loan classification system developed using Python and Scikit learn to analyze applicant data and predict loan approval outcomes. The project focuses on data preprocessing, visualization, classification models, and performance evaluation.

---

# Features

- Loan approval prediction
- Data preprocessing and cleaning
- Missing value handling
- Outlier detection and analysis
- Exploratory data analysis
- Data visualization using plots
- Classification model training
- Model accuracy evaluation

---

# Tech Stack

## Programming Language
- Python

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit learn

---

# Machine Learning Algorithms

- Logistic Regression
- Decision Tree Classifier

---

# Project Workflow

1. Import dataset
2. Data cleaning and preprocessing
3. Handle missing values
4. Perform exploratory data analysis
5. Detect and manage outliers
6. Train classification models
7. Evaluate model performance
8. Predict loan approval outcomes
